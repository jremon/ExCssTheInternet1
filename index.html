Clients and Servers

Computers and servers communicate by establishing a two way connection in which two agents interact:
  • The client, that performs a request of content based on it's needs, through the client-side software that enables him to access to that content
  • The server, that provides the client with the requested data

So Actually What Happens?

  1. The browser is the bridge that makes this interaction possible on the client's edge
  2. The exchange of information takes place through a http request, which asks for a copy of the data hold by the server
